# Flight Management System

## Description
This project is a comprehensive Flight Management System developed using Python and MySQL. It aims to streamline the processes of booking flights, viewing available flights, and managing ticket information, ensuring a seamless and user-friendly experience.

### Features
- **Home**: Main entry point with navigation to all system features.
- **View Flight Details**: Display detailed information about available flights.
- **Book a Flight**: Functionality to book flights with ease.
- **View Ticket**: Access and manage booked tickets.
- **Exit**: Safely exit the application.

### Tools & Technologies
- **Languages**: Python
- **Database**: MySQL
- **Development Tools**: Visual Studio Code, Notepad++
